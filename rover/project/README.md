## 生成AIでROS2ノードを作るチュートリアル - AruCoマーカーに追従するロボットを作る

生成AIを使ってかんたんなROS2ノードを作成してみましょう。
`project` ディレクトリにAruCoマーカーに追従する制御を行うROS2ノード `aruco_trace.py` を作ってみます。

### 推奨環境

- Visual Studio Code (VScode)
  - Github Education を有効化したアカウントでログインし、Copilotを有効化していること

### 手順

1. Copilot を Agent モードで立ち上げる
2. 例えば、以下のように指示してみましょう

    ```txt
    AruCoマーカーのid=0を見つけたらそれに向かって走る制御を行うROS2ノード `aruco_trace.py` を作成せよ
    必要なROS2トピックは rover/README.md を参照すること
    ```

3. コードを確認し、実行してみましょう
4. 不具合があれば、再び聞いてみるか自分で読んで調整してみましょう
   - AIに聞きながらコードの構造を理解するのも良いです

### 備考

- Copilotでの生成サンプルを `sample.py` に提示します
- Webブラウザ等でAIを利用する場合は、条件を自分で明記するか `README.md` などの必要な部分をコピー・ペーストしてみてください
